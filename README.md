# CTF Web

Đồ án môn Bảo mật web và ứng dụng.

## Installation

Window cần cài nodejs và mongodb tham khảo [node](https://nodejs.org/en/) và [mongodb](https://www.mongodb.com/what-is-mongodb). 
Sau đó chạy lệnh:

```bash
cd web && npm i
node app.js
```
Linux chỉ cần chạy lệnh sau:
```bash
chmod +x StartServer.sh
sudo ./StartServer.sh
```
